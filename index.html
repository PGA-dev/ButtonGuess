GUESS:

Think of a number between 1-100 and click the blue button when you're ready.

P.S. Click on body text items to remove (GUESS button to add back)

This game demonstrates a classic programming algorithm: binary search. When using the binary search algorithm, the computer will always guess in the middle of the range; thus halving the search area.This algorithm will find the user's number in 7 or less tries every time for our 1-100 search

Original concept is a CodePen by Minae Lee; this version is an adaptation of the original, using some fun CSS with JS fleshed out by PGA.
